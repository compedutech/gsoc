This repository contains pull requests extracted from the Google Summer of Code program from 2020 to 2023. There are three main folders
1. Data- contains the updated data for GSOC. Currently covers 2016 to 2025.
2. Archive2020-2023 - contains the pull request data used in the SIGCISTE 2025 paper. It covers data from 2020 to 2023
3. Scripts- Contains the Python scripts used to extract and analyze the data

There are other subfolders you may find
1. Raw_GSOC_Description - The description for each GSoC project that was scraped directly from the GSoC official web site. The projects have been grouped into the years.
2. gsoc_url - contains all the URLs for the GSOC participants. The links to the pull requests was extracted from the urls
3. Raw_Pull_Data - Pull request information for each participant. If a name (file_name) appears more than once, it means that the participants participated in multiple GSoC programs (e.g 2020 and 2021)

The data is freely available for non-commercial purpose.
